# SolTides Public Memory

This folder is the public, project-specific handoff for SolTides.

Use it when a new Codex or ChatGPT coding session needs to recover the current
project state from the public repo.

Read these files in order:

1. `START_PROMPT.md`
2. `LATEST.md`
3. `HANDOFF.md`

If a private local clone of `SolTides/ai_memory` is also available on the
machine, that can be used as an extra context source after these files.
